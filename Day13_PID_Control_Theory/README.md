## 📚 核心知识总结

### 一、开环控制

开环控制只有控制输出，没有自动反馈。

例如：

```
STM32
↓

PWM

↓

L298N

↓

电机
```

控制器不会主动检测电机实际速度，因此即使受到外界干扰，也不会自动修正。

---

### 二、闭环控制

闭环控制会不断获取系统的实际状态，并根据误差不断修正输出。

整个过程如下：

```
目标速度
↓

PID 控制器
↓

PWM

↓

L298N

↓

电机

↓

编码器测速

↓

实际速度

↓

返回 PID
```

形成一个不断循环的控制过程，因此称为**闭环**。

---

### 三、为什么需要反馈

如果没有反馈：

- 不知道当前速度
- 不知道是否达到目标
- 不知道有没有受到干扰

编码器测得的实际速度会返回控制器，这个过程就是反馈。

---

### 四、为什么固定 PWM 不可靠

固定 PWM 不能保证固定速度。

影响因素包括：

- 电池电压变化
- 电机负载变化
- 摩擦变化
- 外界阻力
- 电机温度变化

因此需要实时测速并不断修正 PWM。

---

### 五、误差（Error）

误差定义：

```
Error = Target - Actual
```

含义：

- Error > 0：实际速度偏慢，需要加速
- Error = 0：达到目标速度
- Error < 0：实际速度偏快，需要减速

---

### 六、比例控制（P）

比例控制公式：

```
Output = Kp × Error
```

其中：

- Error：当前误差
- Kp：比例系数
- Output：控制输出（PWM）

比例控制思想：

- 误差越大，输出越大
- 误差越小，输出越小
- 接近目标后自动减小控制力度

---

### 七、Kp 对系统的影响

#### Kp 太小

- 输出较小
- 响应速度慢
- 接近目标需要较长时间

#### Kp 太大

- 输出过大
- 容易超过目标值
- 误差方向不断反转
- 系统容易震荡

---

## 💡 今日最大的收获

今天真正理解了：

> PID 不是 STM32 的一个外设，而是一种反馈控制思想。

它通过：

```
测量实际值
↓

计算误差

↓

调整 PWM

↓

再次测量

↓

继续修正
```

不断让系统接近目标值。
