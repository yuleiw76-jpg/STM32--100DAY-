# Day09 - 五路红外循迹传感器与舵机偏差控制

## 今日学习目标

- 理解红外循迹传感器识别黑白区域的原理
- 读取五路循迹模块的高低电平
- 学习 GPIO 浮空输入
- 将五路传感器状态转换成 `track_error`
- 根据 `track_error` 控制舵机角度
- 理解全局变量和 `extern` 的用法

## 实验说明

本实验使用五路红外循迹传感器检测黑线的位置，并通过舵机的五种偏转状态展示位置偏差。

本实验暂时不控制完整循迹小车，主要用于学习循迹传感器的数据读取、偏差转换以及 PWM 舵机控制。

## 红外循迹原理

- 红外发射管向地面发射红外光
- 白色表面反射较强
- 黑色表面吸收较强，反射较弱
- 本实验中检测到白色时输出 `1`
- 检测到黑色时输出 `0`
- 不同模块的输出逻辑可能相反，应以实际测试为准

## 引脚连接

根据 `track.h` 中的实际宏定义，五路信号连接如下：

| 传感器输出 | STM32 引脚 |
|---|---|
| OUT1 | PA4 |
| OUT2 | PA5 |
| OUT3 | PA6 |
| OUT4 | PA7 |
| OUT5 | PB0 |

舵机 PWM 使用 TIM1 通道 1，对应 PA8。

## GPIO 浮空输入

五路循迹输入均使用 `GPIO_Mode_IN_FLOATING`。STM32 不启用内部上拉或下拉，引脚电平由外部循迹模块决定。

源码中同时设置了 `GPIO_Speed_50MHz`。在输入模式下，该参数通常没有实际输出速度意义；GPIO 速度主要用于输出模式。

## 五路状态与偏差值

以下关系来自原始 `track_scan()`：

| OUT1～OUT5 | `track_error` | 含义 | TIM1 CCR1 |
|---|---:|---|---:|
| 01111 | -2 | 黑线位于最左侧 | 500 |
| 10111 | -1 | 黑线偏左 | 1000 |
| 11011 | 0 | 黑线位于中间 | 1500 |
| 11101 | 1 | 黑线偏右 | 2000 |
| 11110 | 2 | 黑线位于最右侧 | 2500 |

CCR1 由下面的表达式计算：

```c
TIM_SetCompare1(TIM1, 1500 + 500 * t_error);
```

实际机械角度会受到舵机型号、安装方向和校准范围影响，应以实物测试为准。

## track_scan()

`track_scan()` 读取 OUT1～OUT5 的高低电平组合，并更新全局偏差变量 `track_error`。

```text
五路传感器状态
→ track_scan()
→ track_error
```

当前代码只处理“恰好一路检测到黑线”的五种组合。遇到其他组合时函数直接返回，因此会保留上一次的 `track_error`。

## servo_control()

`servo_control(track_error)` 根据偏差值修改 TIM1 PWM 的 CCR1，从而改变 PWM 高电平时间并控制舵机偏转。

```text
track_error
→ CCR1
→ PWM 高电平时间
→ 舵机角度
```

PWM 初始化调用为：

```c
TIM1_PWM_Init(20000 - 1, 72 - 1);
```

在 72 MHz 定时器时钟下，预分频后计数周期为 1 μs，ARR 为 19999，对应约 20 ms 周期，即 50 Hz PWM。

## 主程序流程

主程序依次初始化 delay、LED、TIM1 PWM 和 TRACK，然后每隔约 10 ms 扫描一次传感器并控制舵机：

```text
初始化 delay、LED、PWM、TRACK
→ delay_ms(10)
→ track_scan()
→ 更新 track_error
→ servo_control(track_error)
→ 循环执行
```

## 全局变量与 extern

在 `track.c` 中：

```c
int track_error = 0;
```

这是变量定义，会真正分配存储空间。

在当前工程的 `main.c` 中：

```c
extern int track_error;
```

这是变量声明，不会再次分配存储空间。`main.c` 可以通过该声明访问 `track.c` 中定义的同一份全局变量。

可以直接修改该变量：

```c
track_error = 2;
```

但不应在另一个源文件中再次定义同名全局变量，否则可能造成重复定义；若在函数内部重新定义同名局部变量，还会遮蔽全局变量。

## 当前代码局限

当前程序主要处理单路传感器检测到黑线的五种简单情况。后续可以继续研究：

- 两路同时检测到黑线
- 全部检测不到黑线
- 多路同时检测到黑线
- 丢线处理
- 保留上一次偏差
- 加权误差
- PID 循迹
- 舵机或左右电机控制

今天学会了把五路红外传感器的高低电平组合转换成位置偏差，并利用该偏差修改 PWM 的 CCR，让舵机展示黑线所在方向，同时理解了全局变量和 `extern` 的跨文件使用方法。
