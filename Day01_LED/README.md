# Day01 - STM32 跑马灯实验

## 今日目标

完成 STM32 第一个实验：跑马灯。

## 实验现象

程序烧录进开发板后，LED 可以按照规律亮灭。

## 今日学习内容

1. delay 延时函数  
   delay 函数可以让单片机等待一段时间，属于阻塞式延时。

2. LED_Init 函数  
   用于初始化 LED 对应的 GPIO 引脚。

3. typedef 的使用  
   typedef 可以给已有类型起别名，例如给结构体类型起一个更短的名字。

4. .c 和 .h 文件的区别  
   .c 文件主要放函数的具体实现；  
   .h 文件主要放函数声明、宏定义、类型定义等。

5. GPIO 推挽输出  
   推挽输出可以主动输出高电平或低电平。

## 今日 BUG
